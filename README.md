# extract_geo_functions


Inputs

1) XS shape file (e.g., .\sfe_tbs_M1_024_vv3\sfe_tbs_M1_024_vv3.\thalweg_5m_40m.shp )
2) DEM (e.g., .\sfe_tbs_M1_024_vv3\sfe_tbs_M1_024_vv3.\sfe_tbs_m1_024_vv3.tif)

Outputs

1) WSE, bed profile and width plots (.\sfe_tbs_M1_024_vv3\sfe_tbs_M1_024_vv3\GVFs\profiles_XX.png and width_XX.png)
2) WSE, bed profile and width series (.\sfe_tbs_M1_024_vv3\sfe_tbs_M1_024_vv3\GVFs\width_XX.xlsx)
3) XS profile (.\sfe_tbs_M1_024_vv3\sfe_tbs_M1_024_vv3\XS\xsect_XX\profile_YY.png)
